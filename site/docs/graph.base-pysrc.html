 12  This module contains Graphine's base graph represenatation. 
 13   
 14  The goal of Graph is to provide a flexible, easy to use, 
 15  somewhat fast graph implementation. It should be seen as 
 16  a firm foundation for later extension, providing all the 
 17  tools a developer needs to create the appropriate data 
 18  structure for their task with only slight modification. 
 19   
 20  Interface summary: 
 21   
 22  To create a new Graph: 
 23   
 24          >>> from graph.base import Graph 
 25          >>> # I need named nodes and weighted edges 
 26          >>> g = Graph(["name"], ["weight"]) 
 27   
 28  To add nodes: 
 29   
 30          >>> node_1 = g.add_node(name="bob") 
 31          >>> node_2 = g.add_node(name="agamemnon") 
 32   
 33  To add edges: 
 34   
 35          >>> edge_1 = g.add_edge(node_1, node_2, weight=5) 
 36   
 37  Note that what you're getting back isn't a full 
 38  node or edge, eg: 
 39   
 40          >>> node_1 
 41          1 
 42          >>> node_2 
 43          2 
 44          >>> edge_1 
 45          -1 
 46   
 47  As you can see, its just an id used to uniquely 
 48  identify it to the graph. To get the full object, 
 49  just ask nicely: 
 50   
 51          >>> g[node_1] 
 52          Node(name="bob") 
 53   
 54  The same thing works for edges: 
 55   
 56          >>> g[edge_1] 
 57          Edge(start=1, end=2, weight=5) 
 58   
 59  Notice again that it uses those uids as reference 
 60  points. Don't forget them! 
 61   
 62  To iterate over all the nodes in a graph: 
 63   
 64          >>> for node in g.get_nodes(): 
 65          >>>     print(node) 
 66          Node(name="bob") 
 67          Node(name="agamemnon") 
 68   
 69  And, for edges: 
 70   
 71          >>> for edge in g.get_edges(): 
 72          >>>     print(edge) 
 73          Edge(start=1, end=2, weight=5) 
 74   
 75  Of course, if you want to get the uids, just use get_node_uids 
 76  or get_edge_uids. 
 77   
 78  Traversals are just as simple: 
 79   
 80          >>> for node_uid in g.depth_first_traversal(node_1): 
 81          >>>     print(g[node_uid]) 
 82          Node(name="bob") 
 83          Node(name="agamemnon") 
 84   
 85  and similar for depth first traversals.
