Class Edge

source code

GraphElement --+
               |
              Edge

Base edge representation.

Edges have two properties, start and end, which are Node objects incident to the edge.

Edges also provide the key function, which gives a view of the edge suitable for data-centric comparison with other nodes.

They also inhierit the data property, which provides dictionary access to all the non-private portions of the edge.
